# Calendar Tool Demo
## Python简易万年日历工具
控制台日历小工具，实现闰年判断、月份天数查询，练习条件判断、取模运算、列表成员判断、自定义函数封装。

## 项目功能
1. 闰年检测：输入年份自动判断平闰年，解释2月天数差异
2. 月份天数查询：输入年份+月份，自动算出当月总天数（自动区分2月平闰年）
3. 输入校验：限制年份正数、月份1-12区间拦截非法数字
4. 异常捕获：非数字输入不会导致程序崩溃
5. 模块化封装：拆分闰年判断、天数计算两个独立函数，代码整洁易复用

## 运行方式
1. 将代码保存为 calendar_tool.py 直接运行
2. 菜单输入序号选择功能，0退出程序

## 核心练习知识点
1. 取模运算 %：判断整除，实现闰年核心逻辑
2. 逻辑运算符 and / or 组合多条件判断
3. 列表 in 判断：快速区分大月、小月
4. 自定义函数 def 封装重复逻辑，return返回结果
5. while True 循环菜单 + elif 多分支选择
6. try-except 捕获数字输入异常
7. 数值区间校验，过滤非法年月
